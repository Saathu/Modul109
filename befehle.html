Git Cheatsheet

Getting Started

  • git init: Initializes a new Git repository
  • git clone <repository>: Creates a copy of an existing Git repository
  • git config: Sets up Git configuration options

Making Changes

  • git status: Shows the status of files in the repository
  • git add <file>: Adds changes to a file to the staging area
  • git commit: Commits changes to the repository
  • git reset <file>: Unstages changes to a file

Branches

  • git branch: Lists all branches in the repository
  • git checkout <branch>: Switches to a different branch
  • git merge <branch>: Merges changes from a different branch into the current branch

Collaboration

  • git remote add <name> <url>: Adds a new remote repository
  • git fetch <remote>: Downloads changes from a remote repository
  • git pull <remote> <branch>: Downloads changes from a remote repository and merges them into the current branch
  • git push <remote> <branch>: Uploads changes to a remote repository

For more information, see the official Git documentation: https://git-scm.com/docs
